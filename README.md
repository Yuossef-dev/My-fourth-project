# ⌚ Tick - أناقة في إيدك تكنولوجيا في حياتك

موقع ويب مودرن، فخم ومتجاوب بالكامل (Fully Responsive) متخصص في عرض الساعات الذكية والكلاسيكية الفاخرة، يجمع بين الأناقة وتجربة المستخدم السلسة.

---

## ✨ المميزات (Features)

* **هوية بصرية فخمة:** استخدام متناسق للألوان الراقية (الأخضر الغامق `#1A3932` مع الذهبي `#C5A059` والبيج)، المناسبة لبراند ساعات فاخر.
* **متجاوب تماماً مع الشاشات (Responsive Design):** بفضل استخدام **CSS Grid** والـ **Media Queries**، تترتب كروت الساعات تلقائياً لتناسب شاشات الموبايل والتابلت والكمبيوتر.
* **أزرار تواصل مودرن:** أزرار التواصل الاجتماعي مصممة على شكل أيقونات دائرية تفاعلية مع تأثيرات حركية (Hover Effects).
* **تأثيرات حركية تفاعلية:** كروت المنتجات ترتفع بسلاسة وتظهر بـ Box Shadow مميز عند مرور الماوس عليها (`transform: translateY`).

---

## 📂 أقسام الموقع (Sections)

1. **الهيدر (Header):** يحتوي على اسم البراند **Tick** وقائمة تنقل مدعومة بالأيقونات (الرئيسية، منتجاتنا، تواصل معنا).
2. **القسم الأول (Hero Section):** واجهة ترحيبية بسيطة وجذابة تحمل شعار وسلوجان البراند مع زر "Learn More".
3. **معرض الساعات (Products Section):** شبكة ذكية (Grid) تعرض 12 كارت لأشهر الساعات العالمية (مثل Smart Watches وساعات Rolex الكلاسيكية).
4. **قسم التواصل (Contact Section):** يحتوي على أزرار دائرية أنيقة ومباشرة لمنصات التواصل الاجتماعي.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **HTML5:** لبناء الهيكل وتوزيع العناصر والأقسام.
* **CSS3:** للتنسيق الكامل، وعمل تأثيرات الـ Hover، والـ Grid/Flexbox Layouts.
* **Font Awesome v6.5.1:** للأيقونات المستخدمة في قائمة التنقل وأزرار التواصل الاجتماعي.
* **Google Fonts:** خط `Edu AU VIC WA NT Guides` الذي يعطي لمسة جمالية وعصرية للنصوص.

---

## 🚀 طريقة التشغيل (How to Run)

1. قم بتحميل ملفات المشروع على جهازك.
2. تأكد من وجود ملف الـ CSS باسم `moo.css` وملف الـ HTML باسم `moo.html` في نفس المجلد.
3. افتح ملف `moo.html` في أي متصفح ويب (مثل Google Chrome أو Microsoft Edge) واستمتع بالتجربة!
